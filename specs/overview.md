# Overview

Phase 2 of the Hackathon II project transforms the Phase 1 console-based Todo application into a modern, full-stack, multi-user web application.

## Goals
- Multi-user support with secure authentication.
- Persistent storage using a cloud PostgreSQL database.
- Modern responsive UI.
- RESTful API for task management.
- AI-powered chatbot interface for natural language task management.

## Tech Stack
- **Frontend**: Next.js 16 (App Router), TypeScript, Tailwind CSS, ChatKit.
- **Backend**: FastAPI (Python), SQLModel ORM, Google Generative AI (Gemini), MCP Server.
- **Database**: Neon Serverless PostgreSQL.
- **Auth**: Better Auth (JWT-based).
- **Package Management**: UV (Backend), NPM/PNPM (Frontend).
