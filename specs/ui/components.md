# UI Components

## Shared
- `Navbar`: Links for Tasks, Login, Logout.
- `Button`: Standard Tailwind-styled buttons.
- `Input`: Controlled input fields.

## Task Specific
- `TaskItem`: Individual task display with toggle and delete buttons.
- `TaskList`: Container for list of tasks.
- `TaskForm`: Form for creating/editing tasks.
- `EmptyState`: Shown when no tasks exist.
